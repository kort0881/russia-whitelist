## Что такое "белые списки"?

С 2025 года при шатдаунах мобильного интернета (дроны, ЧС) включается **DPI-фильтр**:  
- Разрешены только домены из списка Минцифры  
- VPN, Telegram, YouTube — блок  
- Даже **Wi-Fi не спасает**, если трафик идёт через мобильный шлюз
